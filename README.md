# 培训班管理系统——收费版

《培训班管理系统——收费版》是一款专业的培训班管理收费系统，广泛适合培训班，培训中心，培训学校等场合使用。系统包含主要功能： 基本信息：课程设置、校区设置、教师设置、班次设置、单位设置 学员管理：学员信息 查询统计：欠款统计、期间收款、招生老师查询、学员选课明细

# 如何运行本系统?

本系统基于宏达数据库信息管理开发平台开发，下载后，双击文件夹中"dbimp.exe(或Hadp.exe)"文件即可完整运行对应的信息管理系统。
